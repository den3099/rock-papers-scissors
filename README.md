# rock-papers-scissors
## Introduction
This is an implementation of "<a href="https://www.theodinproject.com/lessons/foundations-rock-paper-scissors">Project: Rock Paper Scissors
</a>" from "<a href="https://www.theodinproject.com/">The Odin Project</a>" curriculum. This project is a simulation of the "Rock Paper Scissors" game, when the user enters the page, a prompt will appear asking to select between rock, paper, or scissors. The user will need to write the answer in the prompt and press enter. If no option is written, if it is a different word, or if it is written wrong, the round will be passed and the prompt for the next round will be displayed. The game finishes when either the player or the computer reaches 5 points.
