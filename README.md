# rock-papers-scissors
## Introduction
This is an implementation of "<a href="https://www.theodinproject.com/lessons/foundations-rock-paper-scissors">Project: Rock Paper Scissors
</a>" from "<a href="https://www.theodinproject.com/">The Odin Project</a>" curriculum. This project is a simulation of the "Rock Paper Scissors" game, the game starts when the user push the "Start Game" button, then the user pushes the "Rock", "Paper" or "Scissors" button and the round result will be shown below the buttons with, also, the option the player selected and the computer's option. The game finishes when either the player or the computer reaches 5 points.
